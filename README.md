## About this micropublication

This repository contains the published version of the Impact Scholars Program
micropublication *"Neuro-Modulating Architecture Priors (NMAP) for Context-Dependent Reconfiguration of Locomotory Circuits in Caenorhabditis elegans"*. The manuscript source is written in
[MyST Markdown](https://mystmd.org/) and the rendered PDF/HTML is built from
the files in this archive.
